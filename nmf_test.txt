4. Как запускать Supervised / semi-supervised NMF

Для supervised NMF нужна папка с отдельными нотами:

isolated_piano_notes/
├── A0.wav
├── A#0.wav
├── B0.wav
├── C1.wav
...
└── C8.wav

Команда:

!python "/content/drive/MyDrive/CompactAMTmodels/colab_scripts/04_run_nmf_baselines.py" \
  --dataset-root "/content/drive/MyDrive/maestro-v3.0.0" \
  --output-dir "/content/drive/MyDrive/CompactAMTmodels/nmf_results" \
  --isolated-notes-dir "/content/drive/MyDrive/CompactAMTmodels/isolated_piano_notes" \
  --mode supervised \
  --split test \
  --sample-rate 8000 \
  --n-fft 2048 \
  --hop-length 512 \
  --nmf-components 88 \
  --nmf-max-iter 400 \
  --relative-threshold 0.2 \
  --top-k 5 \
  --smooth-window 3 \
  --min-active-frames 2 \
  --onset-tolerance 0.05
